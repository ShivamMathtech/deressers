## Getting Started

To run this project locally:

### Prerequisites

- [Node.js](https://nodejs.org/) (version 16+ recommended)
- [npm](https://www.npmjs.com/) (or [yarn](https://yarnpkg.com/))

### Install Dependencies

```bash
npm install
# or
yarn install
```

### Start Development Server

```bash
npm run dev
# or
yarn dev
```

The site will typically be available at `http://localhost:5173` (check your terminal output).

## Deployment

This project can be deployed to platforms like [Vercel](https://vercel.com/), [Netlify](https://www.netlify.com/), or any static hosting that supports Vite/React builds.

- Build for production:
  ```bash
  npm run build
  ```
- Preview locally:
  ```bash
  npm run preview
  ```
